# simple swipe slider
a simple swipe slider basic on jquery,run in mobile(include weixin) and PC

##测试环境：
--PC：chrome 47、firefox 40、IE9+

--mobile:魅蓝note自带浏览器及微信6.3.8、iphone5(ios 9.2)自带浏览器及微信6.3.8
